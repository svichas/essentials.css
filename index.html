Installization

There are some ways to get started. First, see all download options available below, then choose the most suitable option for your need.

  • Install with npm: npm install essentials.css
  • Download latest release

Quick start

Incase you are looking to quickly add essentials.css to your project use CDN option (Copy-paste the stylesheet <link> into your <head> before all other stylesheets to load our CSS.).

<!-- Open-sans font optional -->
<link href="https://fonts.googleapis.com/css?family=Open+Sans" rel="stylesheet">
<!-- Include essentials.css -->
<link href="https://cdn.rawgit.com/svichas/essentials.css/0e5596c0/dist/essentials.min.css" rel="stylesheet" type="text/css">
